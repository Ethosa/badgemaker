# Badgemaker
The Nim badgemaker tool.


## Install
-   via git: `nimble install https://github.com/Ethosa/badgemaker`


## Standart badge params
-   `style` -- "flat".
-   `label` -- "".
-   `value` -- "".
-   `label_text_color` -- "white".
-   `value_text_color` -- "black".
-   `label_color` -- "#212121".
-   `value_color` -- "#e0e0e0".
-   `font` -- "DejaVu Sans,Verdana,Geneva,sans-serif".
-   `font_size` -- 12.
-   `width` -- 120.
-   `height` -- 20.
-   `image_path` -- "".
-   `image_color` -- "".
